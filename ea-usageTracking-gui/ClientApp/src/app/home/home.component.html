Welcome to the Usage Tracking Service

View usage and users, as well as control events.

On-boarding process:

  • Apply to EA team who will set your app up with a client id and secret.
  • Use these credentials to login into Usage Tracking Service.
  • Create your application and add events.
  • Integrate your application and start posting usage events for your users.
  • Use this GUI to track usage events and users.
